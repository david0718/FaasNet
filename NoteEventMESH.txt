Créer un bridge entre deux runtime
Le protocol :
* HeartbeatRequest
* BridgeRequest: username, password
* BridgeResonse

Quand un client fait la souscription alors :
* Créer une session sur un autre noeud.
* Créer la souscription sur un autre noeud.
* Ecouter les message ASYNC_MESSAGE_TO_CLIENT et transmettre vers toutes les sessions actives.
